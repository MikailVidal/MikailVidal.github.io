Data, to me, is more than just numbers—it’s a powerful tool for uncovering insights and telling stories that can drive meaningful change. In my studies, I focus on exploring data, building machine learning models, and drawing insights from complex datasets. My curiosity, paired with a strong technical foundation, is my driving force as I work toward shaping a future where AI and data-driven insights have a transformative impact.

Projects

Some of the work I've done so far

Speech Emotion Recognition model I created a model that listens to audio clips and recognizes emotions like happiness, sadness, or anger. I used special techniques to analyze sound patterns, and trained the model with a dataset of voices expressing different emotions.
Gender and Age Detection I built a program that can guess a person’s gender and age group just from their photo. It uses a face-detection tool to focus on the face, then makes predictions based on patterns it learned from a large set of labeled face images.
Movie Recommendation System (IMDb) I developed a movie recommendation tool that suggests films you might like based on your past choices and movie details. It uses data from IMDb to match users with movies that have similar genres, actors, or ratings.
Real Estate Machine Learning Project For House Price Prediction I created a model that estimates house prices based on features like location, size, and number of rooms. Using a well-known housing dataset, it helps predict property values by analyzing factors that typically influence house prices.
